Why are we doing this?

Agents are eating software. Cloudflare, Stripe, and others have already shipped agent-driven infra tooling. Twilio's CLI is a 2019 retrofit with no MCP server. Our fastest-growing segment is voice agents, audio streaming, messaging — and it deserves better than tooling built for humans.

This hackathon is Plivo's stake in the ground. We're building the tooling layer that makes Plivo native to the agent era — CLIs, MCP servers, debuggers, automations. Tools that an LLM can call directly, not wrappers around human flows.

24 hours. Real code. Real demos. The best projects go to production.
